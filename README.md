# Tic-Tac-Toe
A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.  
This is a two-player browser-based game where players take turns marking X and O on a 3×3 grid.

## 🚀 Features
- Two-player gameplay (X vs O)
- Winner detection logic
- Draw detection
- Game reset button
- Responsive design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
